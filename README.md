# 🎬 Telegram Media Grabber Bot

Бот для скачивания видео и фото из закрытых Telegram-каналов.  
Отправляешь ссылку — получаешь файл, который можно скачать на телефон.

---

## ⚡ Быстрый старт

### 1. Зависимости уже установлены ✅

Python 3.12 и Pyrogram установлены. Если нужно переустановить:
```bash
py -3.12 -m pip install pyrogram
```

### 2. Получи API ключи

1. Перейди на [my.telegram.org](https://my.telegram.org)
2. Войди с номером телефона
3. Нажми **"API development tools"**
4. Создай приложение (название любое)
5. Скопируй **App api_id** и **App api_hash**

### 3. Создай бота

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`
3. Придумай имя и username
4. Скопируй **токен бота**

### 4. Узнай свой Telegram ID

1. Открой [@userinfobot](https://t.me/userinfobot) в Telegram
2. Нажми **Start**
3. Бот пришлёт твой **ID** (число)

### 5. Заполни config.py

Открой `config.py` и заполни:

```python
BOT_TOKEN = "123456:ABC-DEF..."      # Токен от BotFather
API_ID = 12345678                     # api_id с my.telegram.org
API_HASH = "abcdef1234567890abcdef"   # api_hash с my.telegram.org
ALLOWED_USERS = [123456789]           # Твой Telegram ID
```

### 6. Авторизуйся (ОДИН РАЗ)

Двойной клик по `auth.bat` или в терминале:
```bash
py -3.12 create_session.py
```

Введи номер телефона → введи код из Telegram → готово!  
Файл сессии сохранится, повторно вводить не нужно.

### 7. Запусти бота

Двойной клик по `start.bat` или в терминале:
```bash
py -3.12 bot.py
```

---

## 📖 Как пользоваться

1. Открой чат с ботом в Telegram
2. Скопируй ссылку на сообщение из канала
3. Отправь ссылку боту
4. Бот скачает и пришлёт файл — нажми чтобы сохранить!

**Поддерживаемые ссылки:**
- `https://t.me/c/123456/789` — приватные каналы
- `https://t.me/channel_name/789` — публичные каналы

**Поддерживаемые медиа:** видео, фото, GIF, документы, голосовые, аудио, видеосообщения, стикеры.

## 📁 Структура файлов

| Файл | Описание |
|---|---|
| `config.py` | Конфигурация (токены, ключи) |
| `create_session.py` | Авторизация аккаунта (запустить 1 раз) |
| `grabber.py` | Логика скачивания медиа |
| `bot.py` | Основной бот |
| `auth.bat` | Запуск авторизации (двойной клик) |
| `start.bat` | Запуск бота (двойной клик) |
| `requirements.txt` | Зависимости Python |

---

## ⚠️ Важно

- Твой аккаунт **должен быть подписан** на канал, из которого качаешь
- `auth.bat` нужно запустить **только один раз** — дальше бот использует сохранённую сессию
- Бот отправляет медиа как обычные файлы — можно скачать прямо на телефон
