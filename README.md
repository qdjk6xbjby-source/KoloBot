# 🎬 Telegram Media Grabber Bot

Профессиональный инструмент для автоматизированного скачивания медиафайлов из Telegram-каналов (включая закрытые). Бот позволяет обходить ограничения на копирование и сохранять контент напрямую в ваш Telegram.

---

## ✨ Основные функции

- 📥 **Скачивание из закрытых каналов**: Использует User API для доступа к контенту, который недоступен обычным ботам.
- 🔗 **Поддержка любых ссылок**: Работает с приватными (`t.me/c/...`) и публичными (`t.me/username/...`) ссылками.
- 📁 **Все типы медиа**: Видео, фото, GIF, документы, аудио, голосовые и видеосообщения.
- 📦 **Поддержка альбомов**: Автоматически скачивает и пересылает медиагруппы целиком.
- 📊 **Система лимитов**: Встроенная база данных для ограничения количества бесплатных скачиваний для пользователей.
- 🛡 **Безопасность**: Все ключи и токены вынесены в переменные окружения.

---

## 🚀 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что у вас установлен **Python 3.12+**.
Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

### 2. Настройка ключей
1. Получите `API_ID` и `API_HASH` на [my.telegram.org](https://my.telegram.org).
2. Создайте бота через [@BotFather](https://t.me/BotFather) и получите `BOT_TOKEN`.
3. Скопируйте файл `.env.example` и переименуйте его в `.env`.
4. Заполните `.env` вашими данными:
   - `BOT_TOKEN` — токен вашего бота.
   - `API_ID` и `API_HASH` — данные вашего приложения.
   - `ALLOWED_USERS` — ваш Telegram ID (через запятую), чтобы получить безлимитный доступ.

### 3. Авторизация (Один раз)
Для доступа к закрытым каналам необходимо авторизовать ваш основной аккаунт (или специально созданный аккаунт-граббер):
```bash
py -3.12 create_session.py
```
Введите номер телефона и код подтверждения. Файл сессии `user_session.session` сохранится локально.

### 4. Запуск бота
```bash
py -3.12 bot.py
```

## 🛠 Технологии
- **Язык**: Python 3.12
- **Библиотека**: [Pyrogram](https://docs.pyrogram.org/) (Asyncio)
- **База данных**: SQLite (для хранения лимитов и истории)

---

## 📁 Структура проекта
- `bot.py` — Основная логика Telegram-бота.
- `grabber.py` — Модуль для работы с User API и скачивания файлов.
- `create_session.py` — Скрипт для создания файла сессии.
- `config.py` — Загрузка конфигурации из `.env`.
- `database.py` — Работа с базой данных (лимиты пользователей).

---

## ⚠️ Безопасность
**Никогда не публикуйте и не коммитьте файл `.env` и файлы `.session`!** 
Они содержат полные права доступа к вашему аккаунту и боту. Эти файлы автоматически добавлены в `.gitignore`.

---

## 📖 Использование
Просто отправьте боту ссылку на любое сообщение из канала. Бот автоматически определит тип контента, скачает его и перешлет вам.
