# 🎬 Media Grabber Bot

Ваш персональный помощник для скачивания контента из Telegram. Бот позволяет быстро и удобно сохранять видео, фото и другие файлы из любых каналов (включая приватные), даже если там стоит запрет на сохранение.

---

## ✨ Что умеет бот

- 📥 **Скачивание отовсюду**: Сохраняет медиа как из публичных, так и из закрытых каналов.
- 📁 **Все форматы**: Работает с видео, фотографиями, GIF-анимациями, документами и голосовыми сообщениями.
- 📦 **Поддержка альбомов**: Если в сообщении несколько файлов, бот пришлет их все одной группой.
- 📱 **Прямое сохранение**: Все файлы приходят как обычные сообщения — их можно сразу сохранить в галерею телефона или на компьютер.

---

## 📖 Как пользоваться

Пользоваться ботом очень просто:

1. **Найдите пост**: Зайдите в канал, из которого хотите скачать медиа.
2. **Скопируйте ссылку**: Нажмите правой кнопкой на сообщение (или зажмите его на телефоне) и выберите "Копировать ссылку" (Copy Link).
3. **Отправьте боту**: Просто вставьте ссылку в чат с ботом и отправьте её.
4. **Готово!**: Бот скачает файл и пришлет его вам. Вам останется только нажать кнопку «Сохранить».

---

## 💡 Примеры ссылок
Бот понимает любые стандартные ссылки Telegram:
- `https://t.me/username/123` — для публичных каналов.
- `https://t.me/c/12345678/90` — для частных (закрытых) каналов.

---

## ⚠️ Ограничения
- Бот работает только с теми каналами, на которые вы (или бот) подписаны.
- Существует суточный лимит на количество бесплатных скачиваний (проверить статус можно командой `/status`).
